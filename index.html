What is CSS?

CSS is a style sheet language. We use CSS to selectively style HTML elements. For example, this CSS selects paragraph text, setting the color to red: 

p {
    color: red;
}

Ruleset anatomy

Selector

HTML element name at the start

Properties

The aspect of the selector that we want to style

Property value

One of the possible values for a given property

Declaration

A single rule

Errors?

You won't see any errors! CSS fails silently.

Multiple elements

Separate multiple elements by commas. For example:

p,
li,
h1 {
     color: red;
   }

Selector types

Selector name What it selects Example
Element selector All elements of a specified type. p selects <p>
ID selector The element with the specified ID (unique). #my-id selects <p id="my-id">
Class selector Elements with the specified class. .my-class selects <p class="my-class">
Pseudo-class selector Elements in a specific state. a:hover selects links on hover.

Which selector should you use?

Use class selectors. Preferred over ID selectors because they're easier to use - don't need to remember uniqueness.

Exercises

Setup

  • Visit https://github.com/alabhyajindal/css_basics/tree/main/exercises
  • Download index.html and bath.jpg
  • Move both to a folder
  • Open the folder in VS Code

Create and link stylesheet

  • In the same folder, create a new file style.css. Could name it anything. General convention is to name it style.
  • Link the stylesheet to the HTML.
  • In the head tag, enter the following:
    • <link href="./style.css" rel="stylesheet" type="text/css" />

Change the font

body {
  font-family: sans-serif;
}

Center the page

Add styles to the body tag

max-width: 1000px;
margin: 0 auto;

Make the h1 bigger

h1 {
  font-size: 48px;
}

Change the text and background color

Add to body. Check the contrast. 

background-color: #eee;
color: #333;

Position image attribute closer to the image

  • Add id attribution to the p element
  • Give it a negative margin-top of 4px

Increase space between sections

  • Add margin-bottom to section
  • Play around with it - give it a background-color, margin

Remove list dots (or change their style)

The property name is list-style-type. Press Ctrl + Space on the property value area and VS Code will suggest possible values.

Change the link styles

  • Target the a elements
  • Change their color
  • Change their color when it's hovered (hint: pseudo-class!)
  • No matter the styling you choose - links should look visually distinct from other elements. Helps the viewer know it's clickable.
