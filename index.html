What is CSS?

CSS is a style sheet language. We use CSS to selectively style HTML elements. For example, this CSS selects paragraph text, setting the color to red: 

p {
    color: red;
}

Ruleset anatomy

Selector

HTML element name at the start

Properties

The aspect of the selector that we want to style

Property value

One of the possible values for a given property

Declaration

A single rule

Errors?

You won't see any errors! CSS fails silently.

Multiple elements

Separate multiple elements by commas. For example:

p,
li,
h1 {
     color: red;
   }

Selector types

Selector name What it selects Example
Element selector All elements of a specified type. p selects <p>
ID selector The element with the specified ID (unique). #my-id selects <p id="my-id">
Class selector Elements with the specified class. .my-class selects <p class="my-class">
Pseudo-class selector Elements in a specific state. a:hover selects links on hover.

Which selector should you use?

Use class selectors. Preferred over ID selectors because they're easier to use - don't need to remember uniqueness.
